# AirBnB Seattle — Tableau Dashboard

An exploratory data analysis of AirBnB listings in Seattle, WA, focused on pricing patterns, neighborhood distribution, and revenue trends.

👉 [View on Tableau Public](https://public.tableau.com/app/profile/sergio.morales8817/viz/AirBnBFullProject_17797641983030/Dashboard1?publish=yes)

---

## Project Overview

This project analyzes AirBnB listing data from Seattle neighborhoods — primarily Queen Anne, Ballard, and surrounding areas — to surface insights relevant to hosts, investors, and travelers.

**Key questions explored:**
- How do prices vary by neighborhood and property type?
- What is the distribution of room types (entire home vs. private room)?
- How does revenue potential differ across zip codes?
- What does average pricing look like per bedroom count?

## Dataset

**Source:** AirBnB Seattle listing data (scraped January 2016)  
**Records:** ~100 listings across Seattle neighborhoods  
**Key fields:** listing ID, neighborhood, property type, room type, price, host details, latitude/longitude, review scores

## Tools Used

- **Tableau Public** — data visualization and dashboard
- **Excel** — raw data source

---

## Key Insights

- **Queen Anne** dominates listing volume, with a mix of entire homes and private rooms
- Entire home/apartment listings command significantly higher prices than private rooms
- A small number of superhosts account for a disproportionate share of listings
- Pricing clusters are visible geographically, with waterfront-adjacent listings at a premium
